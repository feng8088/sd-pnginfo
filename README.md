# sd-pnginfo
获取stable diffusion图片png绘图参数的工具，可以不需要运行stable diffusion的情况下，获取绘图参数。
这只一个python文件就能够实现，为了方便没有python环境的朋友，所以同时打包了exe可运行文件，运行exe文件即可直接使用。
